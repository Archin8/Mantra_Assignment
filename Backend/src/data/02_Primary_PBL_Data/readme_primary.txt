Primary PBL Data
Use this folder for the core dashboard and program intelligence workflow.
Monthly CSV Files
The csv_exports folder contains one CSV per reporting month:

PBL_School_Response_Data_July_2025.csv

PBL_School_Response_Data_August_2025.csv

PBL_School_Response_Data_September_2025.csv
Each file uses the same translated survey-style columns, plus derived fields for total enrollment, total
attendance, attendance rate, and risk status.
Candidate Notes
Candidates should combine the three monthly CSVs during import if their application needs a single table.
District-level and block-level summaries can be computed from these files.
